6. **★★수익률 헤드라인 = OOS(held-out) 기준만(캡틴 격노 2026-06-28 · ★재위반 2026-06-28 260628_02 "또 나 엿먹이려고 수쓰냐").** 모든 성과 보고의 주인공 숫자는 **held-out OOS**로만. **in-sample·천장(M0~M20 레버최적)·레버최대 수치는 헤드라인 금지** — 꼭 필요하면 `(기간 N개월, in-sample 상한·실전 아님)` 명시 후 보조로만. ★★**모든 수익률 숫자에 예외없이 `(기간, in-sample 상한 / OOS)` 라벨을 붙인다 — 라벨 없는 수익률 노출 = 기만으로 간주(캡틴이 28개월/36개월/OOS 구분 못 해 격노).** ★슬립 방향 명시: 슬립↑이면 수익은 항상 ↓(천장값을 현실값 옆에 나란히 둬 헷갈리게 하지 말 것). 예: REVoi COMBO = **held-out OOS(lev3) +2,121%/MDD−5.5%**가 유일한 '수익률'(천장 +35만%·+322만%·+98018%·+55664%·+6829% 등은 전부 36mo in-sample 천장=헤드라인 폐기). 천장 숫자를 실전인 양 말하면 '사기'.
