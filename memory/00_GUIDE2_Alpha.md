# 지침2 — 알파 상승·검증 지침 ★핵심
(통폐합 2026-06-25. 알파/검증 계열 13개 + 시스템 현황을 그룹화. 검증 무관한 화려한 수치는 전부 '미검증'으로 본다.)

## 그룹 A. 알파 철학 — 단짠 배합 (★출발점)
- 알파는 특별한 **개별 지표 IC가 아니라 '장세(레짐)에 맞는 몇 알파요소의 기계적 배합'**에서 나온다. 단독 IC로 "약하다→기각" 금지(설탕만 쓴 치킨집=폐업 비유, 캡틴 강조).
- 실증: OB·OI·CVD·피보 단독 IC≈0이나 — mom_24h+oi_z 배합으로 reversion +127%, TS+reversion 배합으로 MDD −44%→−9.7%(상관 −0.02), ML 추세+포지셔닝 OOS IC 0.023(단일 0→증폭).
- 적용: ①단독 테스트=참고지 '기각' 아님 ②모든 지표를 재료 카탈로그(`02_Alpha_CheckList/00_AlphaMaterials_Catalog`)에 보존(출처·강한장세·역할·배합후보) ③진짜 알파 = 레짐 판별 → 레짐별 재료 켜고/끄고/가중 ④ML 배합은 OOS로만 신뢰.

## 그룹 B. 검증 시스템 — '진짜 알파인가' 자동판정 (필수 통과)
- **alpha_verification_system.py 3단 판정**: ①알파가능성 = WF(롤링) 부호안정 ≥62.5% (★방향 정반대여도 부호 일관이면 정보 有 — 캡틴 핵심지적) ②엣지확정 = SPRT가 연Sharpe 0.5 검출 ③배포가능 = ① AND CPCV(퍼지+엠바고 15경로) p25>0 AND Deflated Sharpe>0.95 AND 비용후(왕복 8bp) Sharpe>0. 출처=Lopez de Prado(CPCV/퍼지)·SPRT·Bailey(PBO/DSR).
- **첫 검증 3종**: ①룩어헤드(신호는 t직전값만) ②앵커 동치(기존 독립측정과 IC 일치) ③2각도 독립(WF+CPCV). "쉽게 좋은 수치 나오면 뻥" — 실측 1호 Funding_level: 가능성O(WF88%·CPCV p25+0.51)·배포X.
- **★Back2TV(모든 백테 공용 시스템)**: 후보 1개 → ①환각0 검증 → ②**MDD 4단 게이트** 최고수익(M0 무제한 / M30 ≥−30% / M25 ≥−25% / M20 ≥−20%, 각 + 격리마진 강제청산 횟수)을 1회 일괄. (구 ②MDD해제·③MDD−25 2단 → 4단 확장, 캡틴 2026-06-26 CLAUDE.md §26. **M20 통과=챔피언 인증 후보**.)
  - ①검증 = 검증엔진만(§15.1)·**1m 체결 전수겹침**(모든 진입·청산가가 1분봉 [저,고] 내+보유창 실도달)·룩어헤드 규명(피봇=확정봉키·진입가=되돌림 미도달시 base폴백·신호=봉시작 open·shift1·**oi_zscore 롤링z 아핀상쇄 실증 2.4e-9**). 하나라도 실패=폐기.
  - ②사이징 = **격리마진·유지증거금·강제청산 실모델**(hsd=1/lev−mmr−SLIP, mae≤−hsd면 청산). ★선형 환상 금지(레버 과하면 청산=수익 안 늚, 레버30=파산). 레버×증거금 격자서 **4단 각각**(M0/M30/M25/M20) 최고복리 + **강제청산 횟수 의무 산출**(MDD 숫자에 안 잡히는 위험).
  - Pine=v6·임베드≤400(초과=최근분·제목표기)·**사례 6선 필수**(TV 가시범위=임베드 최근분 안에서만·영문/한글 병기 그래프). 단일출처 CLAUDE.md §20·`BACKTEST_OUTPUT_SYSTEM.md`.
