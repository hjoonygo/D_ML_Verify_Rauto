### ① 미래참조·환각 100% 원천차단 검증 (`verify_rev3.py`)
하나라도 실패하면 그 수치는 **폐기**. 통과해야만 ②③로 진행.

- **룩어헤드 규명**:
  - 피봇(눌림목) = `pivots_lr`이 `ph_conf[c+right]`로 **확정봉을 키**로 사용 → 확정 전 미사용.

  - `oi_zscore` = 위에 **롤링z** 적용 → 아핀 불변(`rollz(oi)≡rollz(a·oi+b)`)으로 전표본 정규화가 있어도 **무해**(실측 차 2e-9).

### ②③ 사이징 = 격리마진·유지증거금·강제청산 실모델 (`rauto_paper_engine`)
- `exp = size%/100 × lev`; `mmr`(T1 .004/T2 .005, TIER $50k); `hsd = 1/lev − mmr − SLIP(.0005)`.
- `mae ≤ −hsd` → **강제청산** `p = −exp×(hsd + COST .0014 + |fund|)`, 아니면 `p = R×exp`. **선형 환상 금지**(레버 과하면 청산되어 수익이 안 늚 → 최적 레버가 진짜로 정해짐).
- **레버 × 증거금 격자 스윕**에서 최고 복리 선택 — **4단 각각**: M0 MDD 무제한 / M30 MDD ≥ −30% / M25 MDD ≥ −25% / M20 MDD ≥ −20%. 각 단계 강제청산 횟수 동반 산출(§26).
