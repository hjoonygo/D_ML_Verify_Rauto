# 260626_02_Rauto2_Sys — KeyNote (기술노트 + 알파카드)
세션 2026-06-26. Rauto2 시스템(서버+폰 운영모드) + REVoi 청산향상(R+P70) 챔피언후보 검증·로딩.

## A. 시스템 — Rauto2 (서버+폰)
- **옛 b32 개조 + 신규 5모듈 코어**로 서버(8788)+폰(PWA) 구동. 봇=신호/진입/청산, Rauto=체결·비용·데이터교신(역할분담).
- 신규 엔진: `rauto_live.py`(리플레이/운영 구동기=검증원장 시간순 드러내기·룩어헤드차단·중앙px) + `rauto_datafeed.py`(바이낸스/Dauto 라이브 + 워밍업 빌더 + oi_zscore 라이브계산).
- **운영모드(live)**: 36개월 아님 → **Dauto 워밍업(30일)+실시간 forward**(캡틴 개념정정). 서버 RAUTO2_MODE=live.
- **차트버그 해소**: 봇마다 px따로 → 중앙 px 1개 최상위공유(캔들동일·거래만상이).
- 폰 화면 복원: 챔피언바·체결품질분석·비교카드(자산곡선·장세별PF·주간) + 패닝(꾹누름220ms+거리+관성)·버전칩(r1.0).
- ★oi_zscore 라이브 브리지 해결: rev_side가 oi를 롤링z 재정규화(아핀상쇄) → **누적OI 인과24h롤링z**로 라이브계산해도 앵커 +1851.65% 1원단위 재현(룩어헤드0).

## B. 알파 — REVoi 청산향상 (★전환점·챔피언후보)
- 진단: REVoi 강건(전기간 PF1.49). 약점은 대형손실레짐 없음(본전구간 ER0.15~0.30·강상승숏뿐). 6월 −7.8%=소표본 변동.
- ★진짜 레버 = **청산 '구조 부분익절'(tp_frac)** (진입게이트 아님). 12시나리오 스윕서 압도.
- **R+P70 = REVoi + 부분익절0.7** (레짐스텝 R×1.4는 선택 — R×1.0+P70이 ≥). M20 이빠이 사이징 = **lev6/증거금55%(exp3.3)**.
- 성과(슬립0): **+11,810%/MDD−19.9%/강제청산0** (앵커 tp0 +1851.6%/MDD−24.6% 대비 6배+M20통과).
- §26 4단게이트: M0 lev16 +24억%(MDD−72) / M30 lev8 +124842% / M25 lev5 +38345% / M20 lev6 +11109~11810%. 전구간 청산0.

## C. 검증 (채택검증)
- **CPCV 표준6**(고정config·M20사이징): p25 +253.7%/yr·음수폴드0. (★단 cpcv_std6 MDD위반은 월해상도=낙관.)
- **held-out OOS**: train(2023-05~2025-04)→test(2025-05~2026-04, 6월약세 포함) 거래수준 **+205%/MDD−19.2%/청산0**.
- **민감도**: tp0.5~0.8 · 레짐1.0~2.0 전부 +10000~12780%·위반0% = 칼날 아님.
- **강제청산 검증**: hsd=1/lev−mmr−slip. REVoi 최악역행 −5.1%. 극한레버=청산절벽(lev100 −100% 전소·lev50 −62%), **저레버(lev6 hsd16.2%)=청산0**. §20 선형환상 실증.

## D. ★정직 경계 (과대금지)
- +11,810%는 **전표본 MDD−20에 맞춘 과적합 상한**(캡틴 '이빠이' 요청). 미래보장 아님.
- exp3.3은 **−20 경계**. 더 보수적 held-out 재최적(세션01 cpcv_reopt_exit)은 **MDD−20위반7%·최악폴드−21.4%**. → 실거래 **버퍼사이징(exp~3.0/MDD~−17%) 권장** + 그 1폴드 닫기(tp0.8/소폭트림).
