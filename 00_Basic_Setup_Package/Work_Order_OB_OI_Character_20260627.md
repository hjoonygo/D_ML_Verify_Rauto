# Work Order — OB의 OI 성격 판별 → 진입/청산 로직 완성 (2026-06-27, 캡틴 지시)
> 다음 세션 목표(캡틴 확정 2026-06-27). 스윙매매용. 단일출처=이 파일 + WO 과제4·5(REVoi)와 직결.

## 맥락
- ★캡틴 직관: **OB(유동성이 실제 늘어나는 오더블럭)의 '성격'을 정확히 읽으면** 불리한 진입은 취소하고 유리한 진입은 수량을 늘려 **수익을 크게 올릴 수 있다.**
- ★관건(핵심 난제, 캡틴 정확히 지목): OB에서 **OI가 증가할 때 그게 ⒜신규 숏 진입(매도압력·저지 성격)인지 ⒝롱 청산(또는 신규 롱)인지 OI 단독으론 알 수 없다.** 선물 OI는 롱·숏이 쌍으로 생성/소멸돼 방향이 구조적으로 모호. **이 방향성을 밝히는 것이 관건.**
- 우리 위치: 260627_01 세션 결론=단일 REVoi론 MDD−20 불가. OB-OI 성격판별은 ⒤REVoi 진입품질 향상(불리진입 취소·유리진입 증량) ⒥새 진입신호 후보 둘 다. WO 과제4(OI성격→OB결합)·과제5(돌파실패→저지베팅/돌파→재진입)의 핵심 미해결 조각.

## 과제 (단계별)
1. **★OI 방향성 분해 (핵심 난제)** — OB의 OI증가가 '신규숏 vs 롱청산 vs 신규롱' 중 무엇인지 추정(전부 룩어헤드0):
   - OI변화 × 가격변화 **4사분면**: 가격↑OI↑=신규롱 / 가격↑OI↓=숏커버 / 가격↓OI↑=**신규숏** / 가격↓OI↓=**롱청산**.
   - **CVD(taker buy−sell delta)** 로 '누가 공격적인가' 보강: OI↑ & CVD↓(공격적 매도)=신규숏 / OI↑ & CVD↑=신규롱.
   - 펀딩(롱쏠림), (가능시) **롱/숏 별도 청산데이터**(롱청산 vs 숏청산 at level)로 교차확인.
2. **OB(유동성 구간) 식별** — 유동성 실제 증가 구간(OI↑ + 거래량/구조레벨). POC·구조레벨 결합.
3. **OB 성격 분류 → 저지/돌파 확률** — 그 OB가 저지(rejection)인지 돌파(breakout)인지 OI분해+CVD로 분류. 적중률이 기저(우리 청산맵 자석 51~53%)를 유의하게 넘는가.
4. **진입/청산 로직 완성** — OB 성격 따라: 불리(저지 예상서 역방향 진입)=**진입 취소** / 유리(돌파 확정·방향 일치)=**진입수량↑** / 청산 결정. WO 과제5 양면 통합(돌파실패→저지베팅·돌파확인→재진입 스캘핑).

## 완료조건
- OI 방향분해의 예측력 = 돌파/저지 적중률이 **기저(51~53%)를 유의 상회**(순열검정 p). 못 넘으면 단독신호 아닌 필터/사이징으로 격하.
- 진입/청산 로직 = **CPCV 표준6 전폴드 통과 + held-out + 환각0(1m 전수겹침) + §26 MDD 4단·강제청산 횟수**. 단변량 약하면 §23 단짠 배합(레짐별·사이징/필터 양념).

## 재평가 시점 / ★정직 경계 (필독 — 이미 약하다고 본 부분과 구분)
- ★우리 선행연구(반드시 대조, memory GUIDE2 그룹E·§23): **미시구조(OI·CVD·펀딩·다코인쏠림) 단독 BTC 방향예측은 약함 3차 확인** — oi_zscore IC−0.06(초과 6bp<8bp비용·비단조), 청산맵 자석 51~53%, 다코인 |IC|<0.02. → **단변량 IC로 '약하다→기각' 금지(단짠 배합 철학).**
- ★교훈(이게 이 WO의 근거): **같은 OI 신호도 '적용방식'이 알파를 가른다** — CVD흡수 사이징·OI손절거리 롱전용은 CPCV 통과(채택가치), OI 양측손절은 MDD−21.7% 기각. = OB-OI 방향분해는 **'아직 못 깬' 미탐색 조각**(OI의 롱/숏 모호성 해소 자체를 우리가 정면으로 한 적 없음)이라 가능성 有. 단 단독 과신 금지 → 진입취소/증량·저지/돌파 확률로 적용.
- ★스윙 적합성: 보유 며칠~몇 주 → OB 성격(저지/돌파)이 스윙 horizon서 지속·검증 가능(스캘프 지연 회피). 레짐(추세/횡보)별로 OB 성격 다를 것(레짐 결합).
