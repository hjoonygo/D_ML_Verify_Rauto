# Work Order — 다중알파 + 미시레짐 챔피언선발 → ★매월 수익 + (2026-06-28 캡틴 최종목표)
> 캡틴 비전: 지금의 **REVoi COMBO와 전혀 다른 성격의 알파**를 검색·연구·개발해 **챔피언시스템의 효율을 배가**시킨다. 그래서 **더 디테일한 장세판별(미시레짐) 이론**이 필요하다. **최종목표 = 매월 수익을 +로 만든다.**

## 맥락 (왜)
- 현 챔피언 = **REVoi COMBO = 역추세(평균회귀)형**. 레짐 의존이 극심 — 급락 받아치기·고변동 = 강, **휩소·변동성 죽은 하락 = 약**(최근 5/12~6/28 +19%·하락 PF 0.8).
- 챔피언 자동선발(레짐별 최적봇)도 현재 풀이 **전부 같은 REV 알파 5봇** → 상관 높아 분산효과 0(260627_01 세션 대결론).
- → **전혀 다른 성격(상관 0/음)의 알파**를 더해야 어떤 장세든 누군가는 +를 내고, **디테일한 장세판별**로 그 장세에 맞는 봇을 골라야 **매월 +**가 가능.

## 과제 (단계)
1. **★미시레짐 분류 체계** (= 별 WO '미시레짐 세분'): 휩소율 · 짧은 급변 · 변동성 죽은 횡보 · 점프강도 · 실현변동성 · 추세강도(ADX). HMM(2~3 state) 또는 규칙. 룩어헤드0. (선행연구 HMM·변동성레짐·크립토점프58%·choppy step-aside 검증완료)
2. **★REVoi와 직교(상관 0/음) 다른성격 알파 검색·개발**: 추세추종(돌파·모멘텀) · 변동성(롱/숏 볼) · 캐리(펀딩) · 통계차익(페어) 등. 봇계약 `make_trades`로 제작. 선행연구 조건검색 선행.
3. **★챔피언선발 = 레짐별 최적봇 매핑·자동전환**: 미시레짐 → 그 레짐에 강한 봇 선발(REVoi=급락/고변동, 추세봇=추세, 변동성봇=휩소 등). 결정두뇌(rauto_decision)·관제센터.
4. **★포트폴리오 = 매월 + 검증**.

## 완료조건 (★최종목표 = 매월 +)
- **36개월 전(全) 월 양수**(특히 휩소·변동성죽은 하락월) + 분기별 롱/숏 + **CPCV 표준6 전폴드 통과** + **REVoi와 상관 낮음 입증** + **MDD−20(M20 챔피언인증)**.
- 모든 봇 **환각0**(1m 전수겹침) · 이름표 OOS 정합 · §26 4단·강제청산.

## ★정직 경계 (필독)
- 미시구조(OI·CVD·펀딩·다코인) 단독 방향예측 약함 3차 확인(§23·GUIDE2 E). **다른성격 알파도 단변량 IC로 기각 금지(단짠배합)** — 레짐별 배합·OOS만 신뢰.
- 레짐 세분·다중봇 = **과적합 위험**(WHIP_soft 전례: in-sample 천장이 OOS서 사라짐). **held-out·CPCV 표준6 필수**. 레짐 전환구간 휩소("변동성 깜빡임")도 선행연구 경고.
- "매월 +"는 어렵다 — 단일 알파로는 불가 확정(3차). **상관 낮은 다중알파 + 레짐 선발**만이 유일한 구조적 길.

## ★다음 세션 지시 (2026-07-01 캡틴 — 260629_01 세션 마무리 시)
동기(실증): 라이브 현황에서 **현재 레짐='하락(down)'인데 7일 실현이 챔피언 −2.24%·최고 −1.67%**. 원인 = 이번 하락이 REVoi가 버는 '급락'이 아니라 **7일 −3.21%짜리 약한·휩소성 미끌림**인데 7일추세 분류기가 급락과 한 칸으로 묶음(이름표 +39% 상한 vs 실제 −2%). + **워밍업 46일(40일+)에 챔피언 +8.6%(in-sample forward·OOS아님)** = 캡틴 판단 "40일+ 거래에 8% 미만 = 문제 큼".
1. **★실시간 측정 가능 범위의 미세레짐 분류** — 과제1을 **'라이브에서 인과적으로 계산 가능한 피처'로 한정**. 룩어헤드0 + 미래봉0 + 긴 히스토리 불요(워밍업 범위). 후보(전부 causal·rolling·shift): 실현변동성(ATR/수익률 std) · 추세강도(ADX·회귀 R²·|7일추세|) · **휩소율(최근창 방향전환/로컬반전 빈도)** · 점프강도(대형 1봉 이동 수·크기) · OI충격·펀딩·롱숏쏠림(Dauto 실시간). → 급락/약한하락/휩소하락/저변동횡보/고변동횡보/랠리 등 **REVoi 승(급락) vs 패(휩소·약한하락)를 가르는** 세분.
