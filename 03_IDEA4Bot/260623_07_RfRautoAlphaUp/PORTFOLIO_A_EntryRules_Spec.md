- maxhold 없음(트레일/SL로만 청산). 사이징: size = sat × soi (아래).

## 3. REV 스트림 (가중 80% — 주력)
- **TF**: 8h 봉 (resample 480min, origin=2023-05-01 UTC).
- **신호**(룩어헤드0): 
  · mom_24h = open8.pct_change(3)
  · oi_z = oi_zscore_24h **직전 슬롯 끝값**(`.last().shift(1)`)  ← 룩어헤드 차단
  · zr(x) = x.rank(pct=True) − 0.5
  · **combo = (−zr(mom_24h)) × 0.048 + (−zr(oi_z)) × 0.037**  (IC가중·역추세 방향정렬: 높을수록 매수)

- atr_pct = ((high8−low8).rolling(14).mean() / open8).shift(1)  ← 룩어헤드0.
