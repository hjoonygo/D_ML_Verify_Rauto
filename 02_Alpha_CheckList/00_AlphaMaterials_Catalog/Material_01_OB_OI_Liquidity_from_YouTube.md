## 2. 매매법 전체 (영상 스크립트 분석, 시간 태그)
**① 큰 추세 (출발점, 00:12:47)**: 녹색 **평균단가선**(신규물량 가중평균) 위=롱우위 / 아래=숏우위.
**② 타겟팅 (자석, 00:14:08)**: **OI POC**(신규물량 두꺼운 초록막대) + **고배율 청산물량(100x 빨간선)** 겹치는 곳 = 강력한 자석.
**③ 진입 타점 (리퀴디티 존)**: 가격이 청산물량 흡수하러 내려와 "청산뭉침+OB 결합 = 리퀴디티 존"에서 지지 확인 후 진입.
**익절 (00:15:03)**: 반대편 리퀴디티 존 / 반대편 OI POC 도달, 또는 ChoCH(추세반전).
**손절 (00:17:53)**: FVG 하단 or 직전 저점 이탈. **손익비 1:1.5 이상**.
**보조지표(ICT, 00:09:44~)**: BOS(추세지속)·ChoCH(추세전환)·OB(오더블록)·FVG(불균형갭)·평균단가선.
**예시**: ①매물대 안/밑 롱 → 1.7~2.5% → 위 저항 맞고 하락. ②세력 물량압박 빠른진입 → FVG 손절 → 1:1.5 익절.

## 3. ★재료 특성 (우리 연구와 교차검증)
- **추세(평균단가선)** = 우리 ML OOS 1위 피처 ret_24h와 동일 축. **동영상 고수도 "추세가 출발점"** = 우리 발견 검증.
- **OI POC 자석** = 우리 검증 적중률 53%(가까우면 56%)와 일치 = 약한 자석 실재.
- **청산맵(레버리지별)** = 우리에 없음(OI 총량만). 정밀 불가, 근사만.
- **역할**: 추세=방향 / OI POC·청산=진입 타점(자석) / FVG=손절 / 1:1.5=손익비.

## 6. 최종 검증 결론 (2026-06-23)
- ★청산맵 계산: 우리 데이터로 **가능**(liquidation_model.py 청산가공식 + OI시계열 + 1m가격). 계산법 확보(재료 보존).
  (수집기 dauto_collector는 청산 raw 미수집 = OHLCV/OI/펀딩만. 단 CoinGlass 청산맵도 OI 기반 '계산'이라 우리도 재현 가능.)
- 자석효과: 단순 OI POC 52.6% / 청산 히트맵 POC 51.7% = 둘 다 약함(baseline 50%, OI spike 1864건 6h).
- DN 매매법 기계화 재백테: v1(단순OIPOC) -80% / v2(스윕 12조합) 최선 +2% / v3(청산히트맵) 표본10·-17% = 전부 실패.
- ★판정: 추세(평균단가선)=우리 ML 1위 ret_24h와 교차검증 ✅. 청산 자석 방향예측=약함(51-53%). 동영상 엣지는 '차트 재량(구조·리퀴디티존·세력의도 읽기)'에 있어 단순 기계화 불가.
- 재료 상태: 추세=채택(기존). 청산맵 계산법=보존(미래 재량보조/지지저항 표시용). 청산 자석 방향예측=약함(기각 아님, 약한 재료).
